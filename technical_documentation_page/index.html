Introduction

JavaScript is a programming language that enables you to add interactivity to your website.

It can be used for both client-side and server-side programming.

console.log("Hello, World!");

In JavaScript, you can manipulate HTML elements dynamically.

document.getElementById("element").style.color = "red";
  • Client-side scripting
  • Server-side scripting
  • Dynamic content updates
  • Interactivity in the web page
  • Manipulate DOM elements
Variables and Data Types

In JavaScript, a variable is used to store data values.

let name = "John";

There are various data types in JavaScript such as strings, numbers, and booleans.

const age = 25;
  • String
  • Number
  • Boolean
  • Null
  • Undefined
Functions

Functions in JavaScript allow you to reuse code.

function greet() { console.log("Hello!"); }

You can pass parameters to functions and get return values.

function sum(a, b) { return a + b; }
  • Function Declaration
  • Function Expression
  • Arrow Functions
  • Anonymous Functions
  • Callback Functions
Loops

Loops are used to repeatedly execute a block of code.

for (let i = 0; i < 5; i++) { console.log(i); }

JavaScript supports various types of loops such as for, while, and do-while loops.

while (condition) { ... }
  • For Loop
  • While Loop
  • Do-While Loop
  • For-In Loop
  • For-Of Loop
Objects and Arrays

JavaScript objects are collections of key-value pairs.

let person = { name: "John", age: 25 };

Arrays are ordered collections of values.

let numbers = [1, 2, 3, 4, 5];
